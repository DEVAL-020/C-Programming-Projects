# C Programming Projects

A collection of C programming projects showcasing problem-solving skills, fundamental programming concepts, data structures, algorithms, file handling, and practical applications.

## About

This repository contains various C language projects developed for learning, practice, and academic purposes. The projects range from beginner-friendly applications to more advanced console-based systems and games.

## Projects

| Project | Description |
|----------|-------------|
| Who Wants To Be a Millionaire in C | Console-based quiz game inspired by the famous TV show with scoring system and lifelines. |
| Snake Game in C | Classic snake game with movement, collision detection, and score tracking. |
| User Management System in C | User record management system using structures and file handling. |
| Sudoku Solver in C | Solves Sudoku puzzles using logical algorithms and backtracking techniques. |
| Tic-Tac-Toe in C | Two-player console-based Tic-Tac-Toe game. |
| Snake Water Gun Game in C | Fun game based on the Snake-Water-Gun concept using randomization. |
| Digital Clock in C | Real-time digital clock displayed in the console. |
| Number Guessing Game in C | Guess-the-number game with random number generation. |
| Progress Bar in C | Animated console progress bar implementation. |
| Simple Calculator in C | Basic calculator supporting arithmetic operations. |

## Technologies Used

- C Programming Language
- GCC Compiler
- Standard C Library
- Console/Terminal Interface

## How to Run

1. Clone the repository

```bash
git clone https://github.com/DEVAL-020/C-Programming-Projects.git
```

2. Navigate to the desired project folder

```bash
cd project-folder-name
```

3. Compile the source code

```bash
gcc program.c -o program
```
